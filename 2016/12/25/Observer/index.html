GoF Definition:

Define a one-to-many dependency between objects so that when one object changes state, all its dependents are notified and updated automatically.

概念

在观察者模式里，由一群观察者去观察特定的主体，即被观察者，当其状态发生一些变化时，观察者们将自己注册到被观察者那里以得到即时的通知，同时也可以取消注册。这种模式也被称为发布订阅者模式，通常被用来实时事件处理系统。

例子

微博上明星和粉丝们之间的关系，粉丝关注他们喜爱的明星，就能第一时间获得他们的最新动态，对某位明星失去兴趣时，可以随时取消订阅。
