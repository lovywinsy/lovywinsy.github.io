GoF Definition:

Define an interface for creating an object, but let subclasses decide which class to instantiate. The factory method lets a class defer instantiation to subclasses.

概念

工厂方法模式是一种常用的对象创建型设计模式,此模式的核心是封装类中不变的部分，提取其中个性化善变的部分为独立类，通过依赖注入以达到解耦、复用和方便后期维护拓展的目的。

例子

有两种类型的电视，一种是LED，一种LCD。当其中一个发生了故障，去找维修师傅来修理，他会询问电视损坏程度，屏幕类型等，这样他才能带上对应的修理工具。
