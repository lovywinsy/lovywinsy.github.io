GoF Definition:

Allow an object to alter its behavior when its internal state changes. The object will appear to change its class.

概念

允许一个对象在其内部状态改变时行为也随之改变。

例子

还记得哲学理论中的的螺旋式上升吗。一般而言，当人在经济充裕的时候会玩的多一点，贫穷的时候则更加努力工作，事物在不同状态下会有不同的表现，这些不同的表现又会反过来作用于状态，使其变化。
